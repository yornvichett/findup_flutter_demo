# FindUp Flutter – Project Demo Code

This repository contains a **DEMO version of a real production Flutter project**.

It is shared **for technical review purposes only**, to demonstrate:
- Project structure and folder organization
- Flutter architecture used in a real-world application
- Service / ViewModel patterns
- Async programming and model mapping
- Experience working on a large-scale Flutter project

---

## Important Notice

- This is **NOT a standalone application**
- This code is **NOT intended to be compiled or run**
- Backend APIs, Firebase services, and production endpoints are intentionally replaced with **fake_endpoint**
- No real user data, credentials, or business logic are included

This repository exists only to show **code structure and development experience**.

---

## Project Structure
```
lib/
├─ Core/ # Helpers, services, constants, navigation
├─ Data/ # Models and data handling
├─ Firebase/ # Firebase integration (demo structure)
├─ Pages/ # UI screens and widgets
└─ main.dart # Application entry point (demo)
```
